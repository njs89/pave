# Backend

- folder "simple-server"
- inside the folder run the following commands to...

    - ... install all dependencies (only needed once)
    ```bash
    npm install
    ```
    - ... start the server:
    ```bash
    npm start
    ```


# Frontend

- folder "simple-react"
- inside the folder run the following commands to...

    - ... install all dependencies (only needed once)
    ```bash
    npm install
    ```
    - ... start the server:
    ```bash
    npm start
    ```
    - ... create files for publishing:
    ```bash
    npm run build
    ```
